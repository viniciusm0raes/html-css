Atalhos Visual Studio Code para criação de TAG's aninhadas

nav.menu>ul>li*3>a - (O > aninha e o * determina o número de tags criadas. Usando o TAB, desloca para cada ponto de inserção de dados.)

main>section>article>div.classe>div#test$*3 - (O $ faz uma itenação e cria a quantidade de vezes, identificando unitáriamente cada.)

section*5#secao$>article.conteudo>h1+p{Texto de exemplo}|c - (Como h1 e p são tags de mesmo nível, o + cria as duas em "separado", o texto nas {} inclui o conteúdo de cada parágrafo e o |c comenta cada parte do código criado.)

a:link[href="#"].ligacoes{Opção $}#pg$*5 (Cria 5 links vazios (#), numerados sequencialmente, com ligacoes como classe e um ID "pg" sequencial.)

section.container>div#img$*4>img+a:link[href="#"] (Cria uma section com a classe container, dentro dela 4 divs com ID sequencial. Cada div contendo uma tag de img e um link vazio (#))
