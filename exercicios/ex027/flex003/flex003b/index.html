Flexbox - flex-wrap (column)

Para uma melhor visualização dos efeitos de cada configuração, utilizar o DevTools (F12), ajustar o tamanho da viewport e verificar os deslocamentos dos blocos efetuados por cada propriedade.
Apesar de o código demonstrar a configuração de cada propriedade (flex-direction e flex-wrap) em separado, a W3C indica priorizar o uso do shorthand flex-flow.
A propriedade flex-flow simplifica os dois comandos (flex-direction e flex-wrap) em apenas um. Onde a primeira propriedade é direção e a segunda o encapsulamento.
O flex-wrap (encapsulamento), por padrão está configurado como:
flex-wrap: nowrap;
Configurado como nowrap, os elementos não mudarão de lugar, mas caso sejam maiores que o espaço em tela disponível, gerarão barra horizontal para serem visualizados.
O flex-wrap determina a direção / comportamento dos elementos quando o espaço em tela (viewport) for menor que o elemento pai (container).
A quebra do wrap é sempre no sentido do eixo transversal (cross-axis). Por padrão, de cima para baixo, mantendo o sentido do row (esquerda para direita).
O wrap-reverse mantem a direção da row (esquerda para direita), mas inverte o cross-axis para de baixo para cima, podendo ser também alterado pela mudança de direção da linha pelo row-reverse.
As mesmas configurações são válidas para o column, mas para visualizar os resultados é preciso definir uma altura (height) para o container.
