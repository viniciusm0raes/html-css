flex-direction

Para uma melhor visualização dos efeitos de cada configuração, utilizar o DevTools (F12), ajustar o tamanho do viewport e verificar os deslocamentos dos blocos efetuados por cada propriedade.
As configurações serão aplicadas ao elemento pai (container). Já a propriedade flex: auto; deve ser declarada apenas nos elementos filho. Esta defini que não deve haver espaço em branco no elemento pai. Os elementos filho serão distribuídos (esticados) para ocupar todo o espaço da viewport.
O flex-direction cria 2 eixos (axis) para cada elemento. Cada um deles tendo o seu star e end (direção). Em sua configuração padrão, a linha (row) horizontal, configurará o main-axis (eixo principal) como da esquerda para a direita e o cross-axis de cima para baixo. Já para column, o eixo principal troca de lugar com o eixo transversal (cross-axis) e sua configuração padrão é de cima para baixo e esquerda para direita.
Invertidos (row-reverse ou column-reverse) as direções são alteradas para (da direita para esquerda (row) ou de baixo para cima (column)).
A configuração horizontal padrão do flex-direction é flex-direction: row; (horizontal e da esquerda para a direita) . Para inversão, é utilizada a configuração flex-direction: row-reverse; (da direita para esquerda).
A configuração vertical padrão é o flex-direction: column; (vertical e de cima para baixo). Para inversão, é utilizada a configuração flex-direction: column-reverse; (de baixo para cima).
