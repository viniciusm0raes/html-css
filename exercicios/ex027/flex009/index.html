Flexbox - flex-basis

Para uma melhor visualização dos efeitos de cada configuração, utilizar o DevTools (F12), ajustar o tamanho da viewport e verificar os deslocamentos dos blocos elementos/conteúdos efetuados por cada propriedade.
A propriedade flex-basis se aplica apenas aos elementos filho e age diretamente no main-axis. Ela define um valor padrão para o tamanho deste elemento. Adaptando-se ao conteúdo deste (auto) ou ao valor padrão definido (unidade de medida).O valor definido nesta propriedade é obedecido enquanto o tamanho do container pai permitir sua aplicação.
Os valores disponíveis para a propriedade flex-basis:
auto - valor padrão que define o tamanho do elemento filho conforme o seu conteúdo. Se o tamanho do container pai, for maior que o conteúdo do elemento filho, ele não irá aumentar o tamanho deste para além do necessário, deixando em vazio o espaço restante. Como a configuração testada aqui está como nowrap, se o tamanho do container for menor que o necessário para conter todos os elementos, ele irá reduzir de forma flexível e adaptar o seu conteúdo, conforme o espaço disponível.
unidade de medida - se for definido um tamanho "padrão" para o elemento filho, o elemento não adaptará seu tamanho ao conteúdo, tentará obedecer o valor definido. Caso o espaço do container pai não comporte todos os elementos com esse valor, o conteúdo do elemento filho será quebrado (wrap) para manter o valor definido na propriedade flex-basis. Caso o espaço do container não comporte os elementos, mesmo com seu conteúdo quebrado (wrap), será gerada barra de rolagem.
