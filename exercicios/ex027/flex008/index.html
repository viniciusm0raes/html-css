Flexbox - align-self

Para uma melhor visualização dos efeitos de cada configuração, utilizar o DevTools (F12), ajustar o tamanho da viewport e verificar os deslocamentos dos blocos efetuados por cada propriedade.
A propriedade align-self se aplica apenas aos elementos filho e age diretamente no cross-axis. Como row, o cross-axis é o eixo vertical, enquanto na column, o cross-axis é o eixo horizontal.
Os valores disponíveis para a propriedade alig-self:
auto - é o valor padrão e herda a característica de alinhamento vertical do elemento pai, seja o align-content ou align-items.
flex-start - alinha o elemento no início do cross-axis.
flex-end - alinha o elemento no fim do cross-axis.
flex-center - alinha o elemento no centro do cross-axis.
stretch - faz com que o elemento ocupe todo o espaço do cross-axis.

A (align-self: auto;)
B (align-self: flex-start;)
C (align-self: flex-end;)
D (align-self: center;)
E (align-self: stretch;)
