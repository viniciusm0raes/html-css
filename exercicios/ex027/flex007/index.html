Propriedade order

Para uma melhor visualização dos efeitos de cada configuração, utilizar o DevTools (F12), ajustar o tamanho da viewport e verificar os deslocamentos dos blocos efetuados por cada propriedade.
A propriedade order definirá a ordem de exibição dos elementos. Caso o valor de order seja igual, isso não será definido pelo CSS diretamente, mas sim pela ordem de inclusão no próprio HTML. Um exemplo disso é o caso do A e o F abaixo.
