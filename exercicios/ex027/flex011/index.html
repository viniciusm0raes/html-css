Flexbox - flex-shrink (pode encolher?) / flex-grow (pode aumentar?)

Para uma melhor visualização dos efeitos de cada configuração, utilizar o DevTools (F12), ajustar o tamanho da viewport e verificar os deslocamentos dos blocos efetuados por cada propriedade.
Por pardão, o flex-shrink é 1. Isso significa que o elemento filho pode encolher para caber dentro de seu elemento pai (container). O valor 1 é uma referência de proporção, ou seja, o elemento encolhe na proporção de 1 para 1. Qualquer valor positivo pode aplicar uma proporção diferente para essa configuração.
Também por padrão, o flex-grow é 0. Isso significa que o elemento filho não pode aumentar para ocupar o espaço vazio dentro de seu elemento pai. Aplicando qualquer valor positivo, a proporção seguirá a mesma relação (1/1, 2/2, etc.) em relação ao elemento irmão anterior, não em relação ao elemento pai. Utilizar o DevTools e verificar o comportamento de cada elemento.
Apesar de também funcionarem com o flex-flow como wrap, para uma melhor visualização dos efeitos, a configuração dos elementos pai (containers) foi aplicada como flex-flow: row nowrap.
