Citações completas

Segundo Jeff Noble, no seu livro HTML/CSS para Leigos:

A diferença entre os elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto.

Para esse deslocamento pré-formatado da citação, é usada a tag blockquote.

A tag blockquote possiblita a inclusão do parâmetro cite (blockquote cite="o link aqui") que permite incluir um link do autor, site para compra ou qualquer direcionamento que se deseje para ser observado por buscadores como o Google. Esse parâmetro não retorna nenhuma alteração visual na página.
